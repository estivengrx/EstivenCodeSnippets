# Estiven Code Snippets

A collection of useful code snippets for various programming tasks, organized by language and topic. This repository is designed to *ME* and other developers for quickly finding and reusing code for common scenarios. Project ongoing.

## Project Structure

The repository is organized as follows:

```
EstivenCodeSnippets/
├── Python/
│   ├── Graphs/
│   │   ├── graphs-snippets.py  # Code snippets for creating scientific plots with matplotlib
│   ├── SQL/
│       ├── python-sql-snippets.py  # Code snippets for working with SQL in Python
├── README.md  # Project documentation
```

## Requirements

### General
- Python 3.12 or higher
- Virtual environment for dependency management

### Graphs
- **SciencePlots**: A matplotlib style package for scientific plots.
  - Install it using:
    ```bash
    pip install SciencePlots
    ```
  - **Note**: SciencePlots requires LaTeX to be installed on your system. Install LaTeX using:
    ```bash
    sudo apt install texlive-full
    ```

## Features

- **Graphs**: Create scientific plots using matplotlib and SciencePlots.
- **SQL**: Work with SQL databases in Python.
